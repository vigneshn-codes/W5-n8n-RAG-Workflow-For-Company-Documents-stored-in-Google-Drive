## 🔔 TRIGGER — Google Drive Watchers

Two polling triggers (every 1 min) monitor:
- `fileCreated` → new documents uploaded

Folder ID: configured per environment.

⚠️ Switch to webhook trigger for sub-minute latency.

## Sticky Note 2 — INGESTION PIPELINE
**Canvas Position:** Above the Download → Vector Store insert chain
**Color:** Blue

```
## 📥 INGESTION — Parse, Chunk & Embed

Flow:
1. Download binary file from Google Drive
2. Default Data Loader parses binary → text
3. Recursive Character Text Splitter:
   - chunkOverlap: 100 tokens
4. OpenAI Embeddings → vector representation
5. Insert into Simple Vector Store (key: vector_store_key)

⚠️ In-memory store resets on restart.
Upgrade to Pinecone / Qdrant for persistence.
```
